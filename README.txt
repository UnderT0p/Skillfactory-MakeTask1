В данной директории собраны все необходимые файлы для работы.

Для корректной работы между машиной на которой будет запускаться makefile и локальным сервером необходим настроенный беспарольный ssh доступ.
Также необходим установленный docker на локальном сервере.

Склонируйте репозиторий в домашнюю папку пользователя.(git clone https://github.com/UnderT0p/Skillfactory-MakeTask1)

После отредактируйте файл "usernameAndmachine", а именно впишите после знака "=" [имя пользователя]@[ip адресс сервера], для которого настроен беспароьный ssh доступ

Далее, перейдя в склонированную директорию(cd Skillfactory-MakeTask1/) выполните поочередно три команды:
  make sync - эта команда запустит исполняемый файл wgetScript.sh, результат выполнения - скачанное зеркало сайта в директории ~/Skillfactory-MakeTask1/www.chiark.greenend.org.uk/~sgtatham
  make deploy - эта команда запустит исполняемый файл docker.sh, результат - запущенный docker контейнер c nginx, отвечает по порту :8080.
  make cron - эта команда добавит задачу в расписание, а именно выполнение скрипта update.sh каждую субботу в 03:45 утра.

Все исполняемые файлы содержат описание каждого действия.
